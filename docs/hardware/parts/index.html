Parts Lists for all versions

JSN-SR04T Vertical hardware list

Basic items (same for all versions)

  • 30 pin ESP32
  • 12mm Push Button
  • 0.96 inch OLED Display
  • USB-C Charging Module
  • SD card module
  • Automatic Buck-Boost Step Up Down Module for LiPo usage, 3.3V-version
  • 18650 LiIon Battery
  • SD Card NOTE: Cheap SD cards have caused multiple problems in the first OpenBikeSensor prototypes
  • Power Switch
  • GPS-NEO-6M Module NOTE: consider the large antenna variant (optional in “JSN-SR04T Vertical” but mandatory in “JSN-SR04T Horizontal”), alternative link with antenna only option
  • Cable for display connection

Sensor and pcb related parts

  • 2xJSN-SR04T
  • resistors, capacitors, connectors, etc.
  • PCB v00.03.xx
  • 2xSMD connector - 10pcs, 100pcs

Vertical Case

  • Vertical Case
  • SlideOn Display
  • 4x 20x10x2mm magnets
  • 4x 3x3x3mm magnets
  • 15x M3x8mm DIN912
  • 3x M3x16mm DIN912
  • 5x M3x45mm DIN912
  • 5x M3x30mm DIN912
  • 27x M3 nut DIN934
